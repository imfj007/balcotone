# Balcotone - Premium Men's Clothing Brand

## Project Overview

Balcotone is an innovative web application showcasing a premium men's clothing brand with advanced interactive 3D animations, transformed from the original Fizzi 3D project.

## Branding
- **Name**: Balcotone
- **Philosophy**: Minimalist Masculinity.
- **Aesthetic**: Dark, premium, luxury fashion.

## Tech Stack

- **Frontend Framework**: Next.js 14
- **3D Graphics**: React Three Fiber, Three.js
- **Animation**: GSAP (GreenSock Animation Platform)
- **Styling**: Tailwind CSS
- **State Management**: Zustand

## Key Features

### 1. Interactive 3D Product Showcase
- Custom 3D product cards for clothing items (T-shirts, hoodies, jackets, etc.)
- Advanced scroll-triggered GSAP animations
- Seamless Three.js integration across all sections

### 2. Comprehensive E-commerce Suite
- **Collections Section**: Category-based browsing (Essentials, Outerwear, Loungewear).
- **Product Details**: Interactive sizing and image galleries.
- **Cart & Checkout**: Styled placeholder pages ready for backend integration.

### 3. Responsive & Premium Design
- Fully responsive across mobile, tablet, and desktop.
- High-quality visuals powered by Unsplash.
- Minimalist navigation with `mix-blend-difference` header effects.

## Development Workflow

1. **Plan**: Defined brand identity and asset requirements.
2. **Build**: Implemented custom 3D components and e-commerce routes.
3. **Animate**: Tuned GSAP timelines for a luxury fashion feel.
4. **Deploy**: Optimized and pushed to GitHub.

## Getting Started

### Installation
```bash
git clone https://github.com/bsilvers64/Fizzi.git
cd Balcotone
npm install
npm run dev
```

### Build for Production
```bash
npm run build
```
